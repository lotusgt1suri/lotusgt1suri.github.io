Bootstrap Modal Plugin

A modal is a child window that is layered over its parent window. Typically, the purpose is to display content from a separate source that can have some interaction without leaving the parent window. Child windows can provide information, interaction, or more.

Usage

  1. Via data attributes: Set attribute data-toggle="modal" on a controller element, like a button or link, along with a data-target="#identifier" or href="#identifier" to target a specific modal (with the id="identifier") to toggle.
  2. Via JavaScript: Using this technique you can call a modal with id="identifier" with a single line of JavaScript:
  3. $('#identifier').modal(options)
