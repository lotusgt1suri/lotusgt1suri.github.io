Bootstrap Transition Plugin

The transition plugin provides simple transition effects.

If you want to include this plugin functionality individually, then you will need transition.js once alongside the other JS files. Else, as mentioned in the chapter Bootstrap Plugins Overview, you can include bootstrap.js or the minified bootstrap.min.js.

Transition.js is a basic helper for transitionEnd events as well as a CSS transition emulator. It's used by the other plugins to check for CSS transition support and to catch hanging transitions.

Use Cases

    A few examples of the transition plugin:
  1. Sliding or fading in modals. You can find an example in the chapter Bootstrap Modal Plugin.
  2. Fading out tabs. You can find an example in the chapter Bootstrap Tab Plugin.
  3. Fading out alerts. You can find an example in the chapter Bootstrap Alerts.
  4. Sliding carousel panes. You can find an example in the chapter Bootstrap Carousel Plugin.
