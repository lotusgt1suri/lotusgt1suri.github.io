Striped Progress Bar

To create a Striped Progress Bar:

  1. Add a <div> with a class of .progress and .progress-striped.
  2. Next, inside the above <div>, add an empty <div> with a class of .progress-bar and class progress-bar-* where * could be success, info, warning, danger.
  3. Add a style attribute with the width expressed as a percentage. Say for example, style="60%"; indicates that the progress bar was at 60%.
