Columns Ordering

Another nice feature of Bootstrap grid system is that you can easily write the columns in an order, and show them in another one. You can easily change the order of built-in grid columns with .col-md-push-* and .col-md-pull-* modifier classes where * range from 1 to 11. In the following example we have two-columns layout with left column being the narrowest and acting as sidebar. We will swap the order of these columns using .col-md-push-* and .col-md-pull-* classes.
